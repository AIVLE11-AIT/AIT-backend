## 🛠️ Building the project

Clone the project
```
$ git clone <Repository-URL>
```

Build
```
$ chmod +x ./gradlew
$ ./gradlew build
```

Run
```
$ cd build/libs/
$ java -jar -Xms2048M -Xmx2048M ait-0.0.1-SNAPSHOT.jar
```
